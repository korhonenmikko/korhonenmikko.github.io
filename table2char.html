Jordan blocks of unipotent elements on 2-restricted irreducible representations

Rank 2: A2, C2, G2
Rank 3: A3, C3
Rank 4: A4, C4, D4, F4
Rank 5: A5, C5, D5
Type E6 and E7

Let G be a simple algebraic group over an algebraically closed field K of characteristic 2.

For G of rank at most 5, we list in the tables below for every unipotent element u of G and every rational irreducible 2-restricted KG-module V the Jordan block sizes of u on V.

With Steinberg's tensor product theorem, one can then compute the Jordan block sizes of u on any rational irreducible KG-module.

We also include some computations for simple G of type E6 and type E7.

All of the computations below were done using programs written in MAGMA.

Here are some remarks about the notation, which is mostly standard.

  • Simple roots are ordered by the usual Bourbaki labeling, and we denote the ith fundamental highest weight by ϖi.
  • Highest weight a1ϖ1 + ... + alϖl is also denoted by (a1, ..., al).
  • If the image of unipotent element in a given irreducible representation has Jordan block sizes 0 < d(1) < ... < d(t) and d(i) has multiplicity ni, in the tables we denote this by d(1)n1, ..., d(t)nt.
    • In this notation, the exponent ni is omitted whenever ni = 1.
  • For groups of type Al and the exceptional types, we use the usual Bala-Carter labeling of the unipotent classes.
  • Let G be simple of type Cl, with natural module V and non-degenerate G-invariant alternating bilinear form β on V. In characteristic 2, the conjugacy class of a unipotent element u is labeled by the tuple (d(1)_{ε1}^{n1}, ..., d(t)_{εt}^{nt}), where:
    • The Jordan block sizes of u on V are 0 < d(1) < ... < d(t), with block size d(i) occurring with multiplicity ni.
    • εi equals 0 or 1 for all i, and εi = 0 if and only if β((u-1)d(i)-1v,v) = 0 for all v in the kernel of (u-1)d(i).
  • In characteristic 2, a simple group H of type Dl embeds into a simple group G of type Cl as a subgroup generated by short root subgroups.
    • We label the unipotent conjugacy classes of H in the same way as in G.
    • Here H intersects precisely those unipotent conjugacy classes of G which act with an even number of Jordan block sizes on the natural module.
    • Some unipotent classes split into two conjugacy classes in H, in which case we distinguish them by a prime (') or double prime ('') superscript.
  • In characteristic 2, there exist isogenies between groups of type Bl and type Cl, so the tables for type Bl and type Cl are identical.
